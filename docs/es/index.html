Guia de uso

DLL Icon Forge

Una herramienta de escritorio para crear, editar y reordenar bibliotecas DLL de iconos de Windows a partir de imagenes y archivos ICO.

Entrada
ICO, PNG, JPEG, WebP, SVG, DLL
Salida
DLL de recursos de Windows
Privacidad
Procesamiento local, sin subida

Que puedes hacer

Funciones principales

Crear nuevas DLL

Importa imagenes, revisa vistas previas, corrige recursos no validos y genera una DLL de iconos lista para usar.

Editar DLL existentes

Abre una DLL de biblioteca de iconos, extrae iconos legibles, anade o elimina recursos y guarda una nueva version.

Revisar vistas previas

Muestra los iconos en cuadricula o lista, busca por numero, cambia la paginacion y comprueba la calidad rapidamente.

Recortar y validar

Las imagenes no cuadradas se resaltan. Puedes recortarlas con una seleccion 1:1 antes de guardar.

Operaciones con archivos

Guias rapidas

Crear una nueva DLL

  1. Abre la app y elige Crear.
  2. Arrastra imagenes o seleccionalas desde el disco.
  3. Revisa las vistas previas generadas.
  4. Recorta o elimina las imagenes marcadas en rojo.
  5. Reordena los iconos si es necesario.
  6. Pulsa Generar DLL y elige donde guardar el archivo.

Editar una DLL existente

  1. Abre la app y elige Editar.
  2. Selecciona la DLL de origen.
  3. Revisa los iconos extraidos de la biblioteca.
  4. Anade, elimina, recorta o reordena iconos.
  5. Genera una nueva DLL de salida.
  6. Conserva la DLL original: la app no la sobrescribe directamente.

Compatibilidad

Formatos compatibles

ICO PNG JPEG / JPG WebP SVG DLL

Las imagenes deben ser cuadradas para generar la DLL. Si no lo son, DLL Icon Forge las marca y permite recortarlas.

Seguridad y privacidad

Archivos procesados localmente

DLL Icon Forge funciona en el ordenador del usuario. Los archivos importados no se suben a servidores externos y la app no incluye servicios en la nube ni telemetria.

Windows SmartScreen

Si Windows muestra editor desconocido, se debe a que la app no esta firmada con un certificado comercial. El codigo fuente esta disponible en GitHub.
