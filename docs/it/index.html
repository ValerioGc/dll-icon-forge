Cosa puoi fare

Funzionalita principali

Crea nuove DLL

Importa immagini, controlla le anteprime, correggi le risorse non valide e genera una DLL di icone pronta all'uso.

Modifica DLL esistenti

Apri una libreria DLL, estrai le icone leggibili, aggiungi o rimuovi risorse e salva una nuova versione.

Controlla le anteprime

Visualizza le icone in griglia o lista, cerca per numero, cambia paginazione e verifica rapidamente la qualita.

Ritaglia e valida

Le immagini non quadrate vengono evidenziate. Puoi ritagliarle con selezione 1:1 prima del salvataggio.

Operazioni sui file

Guide rapide

Creare una nuova DLL

  1. Apri l'app e scegli Crea.
  2. Trascina le immagini o selezionale dal disco.
  3. Controlla le anteprime generate.
  4. Ritaglia o rimuovi eventuali immagini evidenziate in rosso.
  5. Riordina le icone se necessario.
  6. Premi Genera DLL e scegli dove salvare il file.

Modificare una DLL esistente

  1. Apri l'app e scegli Modifica.
  2. Seleziona la DLL sorgente.
  3. Rivedi le icone estratte dalla libreria.
  4. Aggiungi, elimina, ritaglia o riordina le icone.
  5. Genera una nuova DLL di output.
  6. Conserva la DLL originale: l'app non la sovrascrive direttamente.

Compatibilita

Formati supportati

ICO PNG JPEG / JPG WebP SVG DLL

Le immagini devono essere quadrate per poter generare la DLL. Se non lo sono, DLL Icon Forge le segnala e permette di ritagliarle.

Sicurezza e privacy

File elaborati in locale

DLL Icon Forge lavora sul computer dell'utente. I file importati non vengono caricati su server esterni e non sono inclusi servizi cloud o telemetria.

Windows SmartScreen

Se Windows segnala autore sconosciuto, il motivo e l'assenza di firma commerciale. Il codice sorgente e disponibile su GitHub.
