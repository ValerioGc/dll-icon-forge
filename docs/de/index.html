DLL Icon Forge

Ein Desktop-Tool zum Erstellen, Bearbeiten und Neuordnen von Windows-Icon-DLL-Bibliotheken aus Bildern und ICO-Dateien.

Zum Repository Anleitungen lesen
Eingabe
ICO, PNG, JPEG, WebP, SVG, DLL
Ausgabe
Windows-Ressourcen-DLL
Privatsphaere
Lokale Verarbeitung, kein Upload

Was moeglich ist

Hauptfunktionen

Neue DLLs erstellen

Importieren Sie Bilder, pruefen Sie Vorschauen, korrigieren Sie ungueltige Ressourcen und erzeugen Sie eine einsatzbereite Icon-DLL.

Bestehende DLLs bearbeiten

Oeffnen Sie eine Icon-Bibliothek-DLL, extrahieren Sie lesbare Icons, fuegen Sie Ressourcen hinzu oder entfernen Sie sie und speichern Sie eine neue Version.

Vorschauen pruefen

Zeigen Sie Icons als Raster oder Liste an, suchen Sie nach Nummern, aendern Sie die Seitengroesse und pruefen Sie schnell die Qualitaet.

Zuschneiden und validieren

Nicht quadratische Bilder werden hervorgehoben. Sie koennen sie vor dem Speichern mit einer festen 1:1-Auswahl zuschneiden.

Dateioperationen

Kurzanleitungen

Eine neue DLL erstellen

  1. Oeffnen Sie die App und waehlen Sie Erstellen.
  2. Ziehen Sie Bilder hinein oder waehlen Sie sie vom Datentraeger aus.
  3. Pruefen Sie die erzeugten Vorschauen.
  4. Schneiden Sie rot markierte Bilder zu oder entfernen Sie sie.
  5. Ordnen Sie Icons bei Bedarf neu.
  6. Druecken Sie DLL erstellen und waehlen Sie den Speicherort.

Eine bestehende DLL bearbeiten

  1. Oeffnen Sie die App und waehlen Sie Bearbeiten.
  2. Waehlen Sie die Quell-DLL aus.
  3. Pruefen Sie die aus der Bibliothek extrahierten Icons.
  4. Fuegen Sie Icons hinzu, loeschen, schneiden oder ordnen Sie sie neu.
  5. Erzeugen Sie eine neue Ausgabe-DLL.
  6. Bewahren Sie die Original-DLL auf: Die App ueberschreibt sie nicht direkt.

Kompatibilitaet

Unterstuetzte Formate

ICO PNG JPEG / JPG WebP SVG DLL

Bilder muessen quadratisch sein, bevor die DLL erzeugt werden kann. Wenn sie es nicht sind, markiert DLL Icon Forge sie und ermoeglicht das Zuschneiden.

Sicherheit und Datenschutz

Dateien werden lokal verarbeitet

DLL Icon Forge arbeitet auf dem Computer des Benutzers. Importierte Dateien werden nicht auf externe Server hochgeladen, und die App enthaelt keine Cloud-Dienste oder Telemetrie.

Windows SmartScreen

Wenn Windows einen unbekannten Herausgeber meldet, liegt das daran, dass die App nicht mit einem kommerziellen Zertifikat signiert ist. Der Quellcode ist auf GitHub verfuegbar.
