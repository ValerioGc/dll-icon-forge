Ce que vous pouvez faire

Fonctionnalites principales

Creer de nouvelles DLL

Importez des images, verifiez les apercus, corrigez les ressources invalides et generez une DLL d'icones prete a l'emploi.

Modifier des DLL existantes

Ouvrez une DLL de bibliotheque d'icones, extrayez les icones lisibles, ajoutez ou supprimez des ressources et enregistrez une nouvelle version.

Verifier les apercus

Affichez les icones en grille ou en liste, recherchez par numero, changez la pagination et controlez rapidement la qualite.

Recadrer et valider

Les images non carrees sont signalees. Vous pouvez les recadrer avec une selection 1:1 avant l'enregistrement.

Operations sur les fichiers

Guides rapides

Creer une nouvelle DLL

  1. Ouvrez l'application et choisissez Creer.
  2. Deposez les images ou selectionnez-les depuis le disque.
  3. Verifiez les apercus generes.
  4. Recadrez ou supprimez les images signalees en rouge.
  5. Reorganisez les icones si necessaire.
  6. Appuyez sur Generer DLL et choisissez ou enregistrer le fichier.

Modifier une DLL existante

  1. Ouvrez l'application et choisissez Modifier.
  2. Selectionnez la DLL source.
  3. Verifiez les icones extraites de la bibliotheque.
  4. Ajoutez, supprimez, recadrez ou reorganisez les icones.
  5. Generez une nouvelle DLL de sortie.
  6. Conservez la DLL originale : l'application ne l'ecrase pas directement.

Compatibilite

Formats pris en charge

ICO PNG JPEG / JPG WebP SVG DLL

Les images doivent etre carrees pour generer la DLL. Sinon, DLL Icon Forge les signale et permet de les recadrer.

Securite et confidentialite

Fichiers traites localement

DLL Icon Forge fonctionne sur l'ordinateur de l'utilisateur. Les fichiers importes ne sont pas envoyes a des serveurs externes, et l'application n'inclut ni services cloud ni telemetrie.

Windows SmartScreen

Si Windows signale un editeur inconnu, c'est parce que l'application n'est pas signee avec un certificat commercial. Le code source est disponible sur GitHub.
